## ГЛАВНЫЙ ВОПРОС (задаёт всем!)

### "Как формируется адрес в браузере?"

**Razor Pages (Лаба 2):**
- `Program.cs` → `app.MapRazorPages()` -- регистрирует все страницы
- Адрес = **имя файла** в папке Pages: `Pages/Product.cshtml` → `/Product`
- Ссылки: `<a asp-page="/Product">` -- tag helper формирует href
- Ключевое слово: **система маршрутизации (Routing)**

**MVC (Лаба 3):**
- `Program.cs` → `app.MapControllerRoute(pattern: "{controller=Home}/{action=Index}/{id?}")`
- Адрес = `/Контроллер/Метод/параметр`
- `/Home/CreateBid` → `HomeController.CreateBid()`
- Ссылки: `<a asp-controller="Home" asp-action="CreateBid">`

**Осипов хочет услышать:** "система маршрутизации", "Routing" и чтобы ты **показал Program.cs**.

## Лаба 1 -- WinForms

### Реальные вопросы Осипова

**Модальные и немодальные окна?**
- Модальное -- блокирует главное окно. Код: `form.ShowDialog()`
- Немодальное -- не блокирует. Код: `form.Show()`

**Калькулятор:**
- Форма не растягивается: `FormBorderStyle = FixedSingle`, `MaximizeBox = false`

**Резиновая форма?**
- **Anchor** -- привязка к краям (Top, Left, Right, Bottom)
- **Dock** -- заполнение стороны (Top, Fill, Bottom)

**Флажок блокирует кнопку -- какой механизм?**
- **Команды (Commands)**, НЕ события! `ICommand`, `CanExecute` определяет доступность

**Триггеры?**
- Автоматическое изменение свойств при условии: `IsChecked=True` → `Background=Green`

### Доп. вопросы

| Тема | Ответ |
|------|-------|
| MDI? | Multiple Document Interface, `IsMdiContainer = true` |
| KeyPress? | `e.KeyChar` -- символ, `e.Handled = true` -- отмена |
| BackgroundWorker? | UI не зависает: `DoWork` -- фон, `ProgressChanged` -- UI |
| UserControl? | Комбинация контролов в одном переиспользуемом компоненте |
| Сериализация? | `JsonSerializer.Serialize(obj)` / `Deserialize<T>(json)` |

### Что показывать
1. Калькулятор -- все операции (sqrt, x^2, 1/x)
2. Форма не растягивается
3. ЛР 6 (графика) или ЛР 7 (async)
4. Ткнуть в код: "вот обработчик Click, вызывает CalcEngine.CalcSqrt()"

## Лаба 2 -- ASP.NET Razor Pages

### Реальные вопросы из записи 28.03

| Вопрос Осипова | Ответ |
|----------------|-------|
| **"Как формируется картинка?"** | `_Layout.cshtml` -- мастер-страница. `@RenderBody()` вставляет содержимое страницы |
| **"Покажите в коде"** | Открыть `_Layout.cshtml` → `@RenderBody()`. Открыть `Product.cshtml.cs` → `OnPost()` |

| SeedData? | Заполняет БД начальными данными при первом запуске |

## Лаба 3 -- ASP.NET MVC

### Реальные вопросы из записи 28.03

| Вопрос Осипова | Ответ |
|----------------|-------|
| **"Из чего состоит рутинг?"** | `{controller}/{action}/{id?}` -- контроллер, action, параметр |
| **"Что стоит за ссылкой?"** | `<a asp-controller="Home" asp-action="CreateBid">` → `/Home/CreateBid` |
| **"Как отдаётся вид?"** | `return View()` -- ищет `Views/{Controller}/{Action}.cshtml` |
| **"Из чего состоит контроллер?"** | Конструктор (DI), action methods (`View()`, `RedirectToAction()`), атрибуты (`[Authorize]`) |
| Разница с Razor Pages? | Razor Pages: файл=endpoint. MVC: контроллер отдельно, view отдельно |
| ViewBag? | Динамический объект для передачи данных в view. `ViewBag.Credits = list` |
| [Authorize]? | Ограничивает доступ: без логина -- нельзя. `Roles="admin"` -- только админ |
| AJAX? | Асинхронный запрос без перезагрузки: `fetch('/Home/BidSearch')` |
| Identity? | Библиотека для авторизации: регистрация, логин, роли |

### Что показывать
1. `localhost:5052` -- таблица кредитов
2. "Подать заявку" → CreateBid + AJAX поиск
3. Логин как `admin@credit.com` / `Admin123!` -- CRUD доступен
4. Без логина -- CreateBid требует авторизации

### Какие файлы открыть

| Когда спрашивает | Файл | Что показать |
|---|---|---|
| "Маршрут?" | `Program.cs` | `app.MapControllerRoute(pattern: "{controller=Home}/{action=Index}/{id?}")` |
| "Контроллер?" | `HomeController.cs` | конструктор + `CreateBid()` + `[Authorize]` |
| "View?" | `HomeController.cs` | `return View()` |
| "AJAX?" | `CreateBid.cshtml` | `fetch('/Home/BidSearch?...')` внизу файла |
