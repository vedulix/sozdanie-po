| ЛР | Тема | Что делать | Время |

| **6** | Scaffold + аннотации + кеш | Делать | 30-50 мин |
| **7** | AJAX + частичные представления | Делать, **адаптировать под ASP.NET Core** | 1-1.5 ч |
| **8** | Авторизация/аутентификация | Делать, **адаптировать под ASP.NET Core** | 1-1.5 ч |

### Проблемы ЛР 7 и 8

**ЛР 7 (AJAX):** PDF написан для старого ASP.NET MVC 5. `Ajax.BeginForm` не существует в Core.
- Решение: `jquery-ajax-unobtrusive` через CDN + `data-ajax` атрибуты, или jQuery.ajax вручную

**ЛР 8 (Auth):** Код использует старый Identity API (`ApplicationUserManager`, `Global.asax`).
- Решение: если проект создан с Individual User Accounts (ЛР 5 обновлённая), Identity уже есть. Добавить seed ролей через `RoleManager<IdentityRole>` + `[Authorize(Roles="admin")]`
